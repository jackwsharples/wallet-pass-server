Place your images here. Minimum recommended:

- icon.png and icon@2x.png (required)
- logo.png and logo@2x.png (required)
- Optional: background.png, thumbnail.png, strip.png (and @2x variants)

Edit pass.json to control colors, logoText, and field layout.
Identifiers (passTypeIdentifier, teamIdentifier, organizationName, description) are replaced at runtime from .env.

When this folder exists, the server uses it instead of the generated placeholders. Restart the server after changes.
